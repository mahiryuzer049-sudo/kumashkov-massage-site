# Runbook

Краткое руководство по запуску, сборке и проверкам.

## Локальный запуск

1. `npm install`
2. `npm run dev`
3. Открыть `http://localhost:8000`

## Сборка

1. `npm run build`
2. `npm run preview`

## Проверки перед релизом

1. `npm run lint`
2. `npm run check`
3. Ручной просмотр ключевых страниц:
   - главная
   - 4 посадочные (`landing-*`)
   - политика конфиденциальности

## Инциденты

Если упала сборка:
- запустить `npm run check` локально
- убедиться, что `node` версии 20+

Если проблемы с отображением:
- очистить кеш браузера
- проверить `public/_headers` на корректность CSP
