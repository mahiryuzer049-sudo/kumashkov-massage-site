# Runbook

Краткое руководство по запуску, деплою и проверкам.

## Локальный запуск

1. `npm install`
2. `npm run dev`
3. Открыть `http://localhost:8000`

## Сборка

1. `npm run build`
2. `npm run preview`

## Проверки перед релизом

1. `npm run lint`
2. `npm run check`
3. Просмотр ключевых страниц:
   - Главная
   - 4 посадочные страницы
   - Политика конфиденциальности

## Инциденты

Если упала сборка:
- Проверить `npm run check` локально.
- Убедиться, что `node` версии 20+.

Если проблемы с отображением:
- Очистить кеш браузера.
- Проверить `public/_headers` на корректность CSP.
