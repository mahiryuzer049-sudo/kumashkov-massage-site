Назначение: единая карта изображений для загрузки/замены без ошибок путей.

## Правило именования

1. Только lowercase + `-`.
2. Формат: `.jpg` для фото, `.webp` как опциональная web-версия.
3. Загружать строго по указанным путям (имя не менять).
4. Если готового фото нет, можно временно ставить нейтральный плейсхолдер.
